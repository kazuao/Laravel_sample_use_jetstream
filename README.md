# Laravel_sample_use_jetstream

## day1
1. .envの修正
2. databaseの作成
3. Jetstreamのインストール
```$ composer requre laravel/jetstream```
4. livewireをインストール
```$ php artisan jetstream:install livewire```
5. マイグレーション
```$ php artisan migrate```
6. JSとCSSをビルド
```$ npm install && npm run dev```
7. Dashboard Bladeを編集する
8. views/components/を作成する
